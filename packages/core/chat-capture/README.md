# @memorylayer/chat-capture

Chat conversation capture and normalization for MemoryLayer.

## Overview

The Chat Capture module provides a unified interface for ingesting chat conversations from different AI providers (ChatGPT, Claude, etc.). It normalizes provider-specific formats into a consistent data structure that can be stored and processed by MemoryLayer.

## Features

- Parse ChatGPT and Claude conversation exports
- Normalize conversations to a unified format
- Extensible parser registry for custom providers
- Support for both file imports and streaming API captures
- Comprehensive validation and error handling

## Installation

```bash
npm install @memorylayer/chat-capture
```
